# URL Shortener CLI

This Python script shortens long URLs using the TinyURL API.

## Features

* Convert long URLs into short links
* Simple command line interface
* Uses TinyURL API

## Requirements

* Python 3
* requests library

Install dependency:

pip install requests

## Run

python url_shortener.py

Enter the long URL when prompted.

## Example

Input:
https://www.example.com/some/very/long/url

Output:
https://tinyurl.com/xxxxx
